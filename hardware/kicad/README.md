# KiCad Hardware Design

This directory contains the KiCad 9 project files for the electronics design.

## Components

This section mentions notable components and links to relevant sources of documentation.

- MCU: ST [STM32L431CCUx](https://www.st.com/en/microcontrollers-microprocessors/stm32l431cc.html)
- Radio LoRa: []()
- Sensors:

# KiCad hardware design

This directory contains the KiCad 9 project files for the electronics design.

- Project file: `sensor-node.kicad_pro`
- Top-level schematic: `sensor-node.kicad_sch`
- PCB layout: `sensor-node.kicad_pcb`

The actual contents of these files are currently placeholders. Open or create
them with KiCad 9 to start the design.

# laura32

The project is a state of the art battery-powered LoRa environmental sensor.

## Repository layout

- `hardware/` – KiCad 9 electronics design and mechanical enclosure
- `firmware/` – Firmware source code and build files (C/C++ with CMake)
- `datasheets/` – Datasheets for key components
- `docs/` – Project documentation (behaviour, design notes, etc.)
